## 💻 About this Repository (CV Landing Page)

This repository holds my personal CV and landing page, built with **Next.js (App Router)**. It implements a warm, editorial design with a vertical experience timeline and automatic PDF resume generation.

### Stack
- Next.js 14 (App Router)
- React 18
- Plain CSS (no framework) with CSS variables and responsive breakpoints
- Google Fonts (Newsreader, IBM Plex Mono)

### Getting started

```bash
npm install
npm run dev
```
Open http://localhost:3000 to view it in the browser.
